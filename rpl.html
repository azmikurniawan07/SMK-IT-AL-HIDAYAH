Pengembangan Perangkat Lunak & Gim/Rekayasa Perangkat Lunak

Kompetensi keahlian yang fokus pada pengembangan perangkat lunak mulai dari pembuatan, pemeliharaan, manajemen organisasi pengembangan perangkat lunak, dan manajemen kualitas.

Tentang Kompetensi Keahlian

RPL merupakan salah satu jurusan komputasi yang sangat diminati di era transformasi digital saat ini. Siswa tidak hanya diajarkan cara menulis baris kode pemrograman (coding), melainkan juga diajarkan bagaimana mendesain arsitektur aplikasi, menganalisis kebutuhan pengguna, dan mengelola proyek pembuatan perangkat lunak berstandar industri.

Dengan pendekatan pembelajaran berbasis produk (*Product-Based Learning*) dan *Teaching Factory*, lulusan diarahkan untuk siap bekerja sebagai tenaga profesional di perusahaan teknologi makro maupun sebagai *technopreneur* mandiri.

Pemrograman Web (HTML, CSS, JS, PHP)
Aplikasi Mobile (Android / Flutter)
Pemrograman Berorientasi Objek
Basis Data (MySQL / PostgreSQL)

Cloud Computing & DevOps Dasar

Prospek Kelulusan / Karier

Full Stack Web Developer

Membangun, mengelola, dan memelihara aplikasi web dari sisi tampilan depan (frontend) maupun sistem database belakang (backend).

Mobile Apps Programmer
